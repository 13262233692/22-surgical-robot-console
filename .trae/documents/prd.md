## 1. 产品概述

远程手术机器人控制台——面向跨地域手术场景的生命安全级远程医疗系统。主刀医生通过该控制台实时操控远端手术机器人的机械臂，并接收腔内双目4K内窥镜视频流与力反馈数据，实现"所见即所触、所触即所动"的远程手术闭环。

- 核心价值：将手术专家的触觉与视觉能力零延迟延伸至千里之外的手术台，解决医疗资源分布不均问题
- 目标用户：三甲医院主刀医生、远程手术操作员

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 主刀医生 | 医院LDAP认证 | 全部操作权限：视频监控、机械臂操控、力反馈感知 |
| 手术助手 | 医院LDAP认证 | 只读视频监控、状态查看 |
| 系统管理员 | 管理员账户 | 系统配置、设备管理、日志审计 |

### 2.2 功能模块

1. **手术控制台主页**：系统连接状态、设备就绪指示、延迟监控、一键启动/停止手术
2. **双目视频监控面板**：左眼/右眼4K视频流独立渲染、3D融合预览、画面参数调节
3. **机械臂操控面板**：三维坐标实时显示、夹爪开合度控制、运动轨迹可视化
4. **力反馈可视化面板**：六轴力/力矩雷达图、碰撞预警、触觉强度映射
5. **系统监控面板**：网络延迟、丢包率、WebRTC ICE状态、WebSocket连接状态

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 手术控制台 | 系统状态栏 | 显示WebRTC/WebSocket连接状态、端到端延迟、丢包率、系统时钟 |
| 手术控制台 | 双目视频区 | 左右眼独立Canvas渲染4K视频流，支持画中画3D融合模式，SDP协商状态指示 |
| 手术控制台 | 机械臂操控区 | 500Hz采样的三维坐标(x,y,z)实时曲线、夹爪开合度(0-100%)仪表盘、运动轨迹3D轨迹图 |
| 手术控制台 | 力反馈雷达图 | 六轴力/力矩(Fx,Fy,Fz,Mx,My,Mz)雷达图实时绘制，碰撞阈值预警线，触觉颜色映射 |
| 手术控制台 | 日志与告警 | 系统事件日志流、网络异常告警、碰撞告警、延迟超限告警 |
| 设备管理 | 连接配置 | 信令服务器地址配置、ICE候选策略选择、WebSocket端点配置 |
| 设备管理 | 触觉手柄校准 | 手柄零点校准、工作空间范围设置、灵敏度调节 |

## 3. 核心流程

### 3.1 手术会话建立流程

1. 主刀医生登录系统，进入手术控制台
2. 配置信令服务器地址，发起WebRTC连接
3. SDP Offer/Answer交换，ICE候选收集与穿透
4. 双目4K视频流建立，左右眼画面渲染确认
5. WebSocket数据通道建立，触觉手柄设备初始化
6. 力反馈通道验证，碰撞检测阈值设置
7. 所有通道就绪，系统进入"手术就绪"状态
8. 主刀医生确认启动，手术会话正式开始

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色手术级UI（#0A0E17 主背景），搭配青色（#00F0FF）作为安全/活跃指示、红色（#FF3B3B）作为告警/危险指示
- **次色调**：深灰面板（#141A2A）、边框高亮（#1E2A3A）
- **字体**：JetBrains Mono 用于数据/数字显示，Source Han Sans 用于中文标签
- **布局风格**：全屏沉浸式单页面，T型布局——顶部状态栏、左侧视频区、右侧操控与力反馈区
- **图标风格**：线性图标，2px描边，医疗/工程风格
- **动效**：数据脉冲动画、告警闪烁、连接状态呼吸灯

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 手术控制台 | 系统状态栏 | 顶部固定栏、呼吸灯连接指示器、数字延迟显示(ms)、丢包率进度条 |
| 手术控制台 | 双目视频区 | 两个16:9 Canvas并排、SDP状态标签、ICE状态灯、画面参数滑条 |
| 手术控制台 | 机械臂操控区 | 三轴坐标数字显示+实时曲线、夹爪开合度圆弧仪表、3D轨迹Canvas |
| 手术控制台 | 力反馈雷达图 | 六边形雷达Canvas、预警阈值虚线、力值颜色热力映射、碰撞闪烁动画 |
| 手术控制台 | 日志告警 | 底部可折叠日志流、告警横幅、分级颜色标记 |
| 设备管理 | 连接配置 | 表单输入框、连接测试按钮、ICE策略下拉选择 |

### 4.3 响应式设计

- 桌面优先（4K显示器为主要目标）
- 最小支持1920x1080分辨率
- 4K显示器下视频区原生渲染，1080P下自适应缩放
- 不支持移动端（手术场景专用设备）

### 4.4 设计理念

- **手术级安全**：所有状态变化必须视觉反馈，0.5秒内告警可见
- **信息密度**：关键数据一屏可见，减少翻页和切换
- **暗色沉浸**：长时间手术不产生视觉疲劳
- **数据精确**：所有数值精确到小数点后2位，延迟精确到整数毫秒
